# Saguaros Brand Guidelines (Derived for Hub)

Last updated: 2026-03-05

## Source Notes
This guideline is derived from public Saguaros site patterns (`saguaros.com`, `about`, `olympiad`) to align tone and visual direction. Do not copy exact site layouts/assets; use this as a system reference for the internal Hub.

## Brand Direction
- Tone: philanthropic, energetic, premium community events.
- Messaging style: mission-first, impact-focused, action-oriented.
- Visual mood: warm desert palette + athletic/event energy.

## Core Palette (App Tokens)
- Forest: `#0f1a13`
- Forest Mid: `#1a3425`
- Saguaro: `#2d5f3f`
- Saguaro Light: `#4a8a62`
- Gold: `#c8932a`
- Gold Bright: `#e4b23e`
- Sand: `#e8dcc4`
- Warm White: `#faf7f2`
- Terracotta Accent: `#c44d2a`

## Typography
- Display: Barlow Condensed (headlines, labels, nav emphasis).
- Body: Sora (body copy, form text, helper text).
- Character: compact display headlines + clear body readability.

## Layout and Components
- Sidebar-first workspace for admin utility pages.
- Soft card surfaces (`bg-warm-white`) with subtle sand borders.
- Dense information modules should use compact spacing and strong section labels.

## Motion
- Use meaningful entrance/stagger transitions for key sections only.
- Keep long-running animation minimal in data-heavy screens.

## Voice and Copy
- Prefer direct language: "Connect", "Sync", "Reconcile", "Review".
- Keep labels operational and short.
- Use outcome-oriented text in cards (what this enables now).

## Imagery and Iconography
- Event imagery: candid, active, community-focused.
- Icons: simple outlined system icons, no novelty packs.
- Avoid overusing decorative graphics in admin workflows.

## Accessibility
- Maintain WCAG AA contrast minimum for all text.
- Ensure keyboard focus states are visible in all nav and form controls.
- Avoid relying on color alone for status indications.
